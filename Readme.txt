This script alerts you once it finds the available slot which is usually random.You can schdule the vaccination once the script alerts you.

Registration:

Register at https://www.cowin.gov.in/ using your phone number and Aadhar.

Environment setup:

After downloading the code from repository, install the required libraries using following 
command while you are inside the directory where the codes exists.

pip install -r requirements.txt   


Parameters:

In cowin_aler.py file, Please specify the following parameters

age ---> Mandatory
Pincode ---> Mandatory
Vaccine ----> optional either ""  or COVAXIN, COVISHIELD, SPUTNIK
